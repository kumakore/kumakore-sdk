Detailed Description

KumakoreApp is the main control structure for all actions related to kumakore. You need to create a KumakoreApp for each instance of an activity to ensure safe execution of network operations. Each KumakoreApp instance will load saved preferences, thus it's not necessary to signin to each instance.
